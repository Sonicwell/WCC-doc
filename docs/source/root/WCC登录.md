## 1.1WCC 登录

在您使用的浏览器地址栏中输入网站地址，回车进入系统登录页面。

用户账号和密码是预先设定好的，输入正确的账号和密码，选择【登录】，即可成功进入系统。用户可选择【记住】，浏览器会记住本次的账号密码，方便下次登录。

（如下图所示）
