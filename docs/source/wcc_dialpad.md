## 1 WCC 网页话机的使用说明

### 1.1注册

用户在Chrome插件市场，输入wcc网页话机，下载安装wcc网页话机插件。

![alt text](_static/images/wccdialpad/image1.png)

安装插件后，填写坐席的服务器地址、用户名 (坐席号码+client域名) 、密码（ 在坐席编辑中查看坐席的密码 ）。 

选择【记住我】表示下次 打开插件会记住当前坐席话机设置。

选择【自动登录】则表示打开插件话机会直接登录到当前坐席的话机页面，无需点击登录。

![alt text](_static/images/wccdialpad/image2.png)

登录页面最下方标注话机的最新版本号，为避免使用时发生故障，请注意更新最新版本话机进行使用。

### 1.2页面介绍

#### 1.2.1 拨号盘页面

登录话机后，首先进入的是拨号盘页面

![alt text](_static/images/wccdialpad/image3.png)

![alt text](_static/images/wccdialpad/image4.png)

- 免打扰：当开启免打扰时，坐席将不会接到呼入的电话，通话记录的挂断原因将显示为DND。  

- 自动应答：当开启自动应答时，有通话呼入，通话可以被自动应答，无需坐席手动点击接听键。

- 暂停：当开启暂停时，坐席页面会显示暂停状态，坐席将接不到呼入电话，通话记录的挂断原因将显示为被叫忙。

- 话后：当通话是发生在预拨号或是呼入到坐席组的情况下，坐席挂断通话后，话机将自动进入话后状态。话后状态只有预拨号和呼入到坐席组时可以触发，无法手动进入话后状态。

- 组：坐席可以点击签入签出坐席组，动态坐席组不可编辑。

![alt text](_static/images/wccdialpad/image5.png)

① 坐席在拨号区可以拨号呼叫

![alt text](_static/images/wccdialpad/image6.png)

② 如图所示按键为 " 共享web状态 " 按键

![alt text](_static/images/wccdialpad/image7.png)

用户可以在设置-->高级设置-->开启共享Web状态，开启后，坐席点击该按钮，会直接跳转至坐席页面  

![alt text](_static/images/wccdialpad/image8.png)

当关闭 " 共享Web状态 " 点击这个按钮，会直接弹出系统今日登录页面状态。

③ 如图所示按键为 " 自定义 " 按键

![alt text](_static/images/wccdialpad/image9.png)

点击 " 自定义按键 " 进入自定义按键的编辑页面  

![alt text](_static/images/wccdialpad/image10.png)

保存后，可以在坐席拨号页面显示，设置的号码可以直接点击呼叫。

![alt text](_static/images/wccdialpad/image11.png)

当直接点击该自定义按键，通话会直接呼出如下图

![alt text](_static/images/wccdialpad/image12.png)

如果设置的是坐席号码，则会出现坐席闪烁灯，以便监控这个坐席的状态。当键值坐席为空闲状态时，灯为绿色；当坐席在进行内线通话或是有呼入到该坐席时，灯为黄色闪烁。

【空闲常规状态下】

![alt text](_static/images/wccdialpad/image13.png)

【有内线通话，或是呼入通话】

#### 1.2.2 通话操作

坐席在拨号盘中拨号，并点击通话按钮进行拨打。电话接通后如下图  

![alt text](_static/images/wccdialpad/image15.png)

图示按钮为保持键，点击保持坐席页面显示保持状态通话进入保持状态，对方听筒播放保持音乐。

![alt text](_static/images/wccdialpad/image16.png)

图示按钮为转接键，坐席可以使用这个按钮进行盲转或是咨询转。

![alt text](_static/images/wccdialpad/image17.png) 

如果选择【转接】按钮，通话被直接盲转到输入的坐席。如果选择【咨询】则会进入咨询页面。

![alt text](_static/images/wccdialpad/image18.png)

当被咨询坐席接通后，坐席可以继续选择咨询转接或是开启三方通话的会议模式

![alt text](_static/images/wccdialpad/image19.png)

#### 1.2.3 最近通话

 坐席可以在最近通话页面查看联络历史记录，这里通话历史记录不做限制。

![alt text](_static/images/wccdialpad/image20.png)

选中一条通话记录双击，可以直接回到拨号盘页面拨打这个号码。

坐席可以在通讯录页面添加内部或是外部的联系人，并按姓名正序的顺序进行排列展示。

![alt text](_static/images/wccdialpad/image21.png)

![alt text](_static/images/wccdialpad/image22.png)

坐席可以使用关键字对通讯录中的联系人进行搜索  

![alt text](_static/images/wccdialpad/image23.png)

#### 1.2.5 设置

##### 1.2.5.1 个人信息

坐席点击设置-->个人信息即可进入坐席个人信息的查看页面

![alt text](_static/images/wccdialpad/image24.png)

在个人信息页面可以查看坐席的头像、分机号 ( 坐席工号 ) 、用户名、部门等在系统中设置的信息，以及坐席登录的部分登录信息如下图所示 。

![alt text](_static/images/wccdialpad/image25.png)

##### 1.2.5.2 高级设置

坐席在高级设置页面可以对通话、网络以及设备进行设置  

![alt text](_static/images/wccdialpad/image26.png)

【通话设置】

- 回声消除：消除通话回声，让通话更加清晰。

- 噪声抑制：消除通话噪音，让通话更加清晰。

- 自动增益：自动调节麦克风输入音量，保持输出音量稳定，让对方听得清晰。

【网络设置】

- 自动注册：打开自动注册时，当坐席登录网页话机，该话机就会自动注册上。

- 共享Web状态：打开该按钮时，坐席可以在拨号盘页面点击按键，一键登录到坐席的仪表盘页面。

- WSS服务器：通过浏览器访问话机的IP地址来配置。

【设备设置】

- 输入设备：选择麦克风。

- 输出设备：选择扬声器。

- 编码设置：包含音频和视频的编码选择设置。

##### 1.2.5.3 弹屏设置

弹屏设置是指坐席在呼入呼出的时候，系统会自动弹出一个指定的地址

![alt text](_static/images/wccdialpad/image27.png)

点击【新增】进入弹屏添加页面。  

![alt text](_static/images/wccdialpad/image28.png)

- 名称：你可以为这个弹屏设置一个名称。

- 启用：你可以选择是否启用这个弹屏功能。

- 同样URL单次弹屏：坐席登录话机后有同样的url，话机检测后将只会弹出一次。

- 呼入弹屏地址：输入一个当有电话呼入时会弹屏的地址。

- 呼出弹屏地址：输入一个当有呼出电话时会弹屏的地址。
