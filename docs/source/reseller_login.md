## 1.1 WCC 登录

1. 访问登录页面

打开主流浏览器（推荐 Chrome 90+、Edge 88+ 版本），在地址栏输入系统官方网址，按下 Enter 键，直接进入 WCC 代理商登录页面。

2. 填写登录信息

账号：使用上级用户预先配置的专属账户（格式以上级告知为准，通常为 “用户名 / 工号 + 关联标识”）。

密码：输入上级用户预设的初始密码（首次登录建议立即修改，提升账号安全性）。

3. 完成登录

若使用个人办公设备（信任设备），可勾选【Remember me（记住密码）】，浏览器将自动保存账号密码，方便下次快速登录。

确认账号密码无误后，点击【Sign In（登录）】按钮，即可成功进入代理商系统后台。

![alt text](_static/images/reseller/media/image1.png)

4. 注销登录

登录系统后，找到页面顶部状态栏的下拉菜单（显示当前登录账号，如 “receller.playwright”）。

在下拉菜单中选择【登出（注销）】选项，弹出确认窗口后点击【确认】。

系统将自动清除当前登录状态，页面跳转回 WCC 登录页面，完成注销操作。

![alt text](_static/images/reseller/media/image2.png)

5. 注意事项

- 公共设备（如共享电脑、网吧设备）请勿勾选【记住密码】，防止账号信息泄露。

- 初始密码仅用于首次登录，登录后需通过【我的档案】修改为高强度密码（建议包含字母、数字及特殊符号，长度不低于 8 位）。

- 若账号长时间未使用，再次登录前可联系上级用户确认账号状态是否正常。

- 输入账号密码后无法登录，可检查网址是否正确、浏览器版本是否适配，或联系上级用户核实账号权限及密码有效性。

- 登录后若出现页面加载不全，可点击状态栏【REFRESH（刷新）】按钮，或清除浏览器缓存后重新登录。
